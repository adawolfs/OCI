# Jugado con Container Runtimes

Para esta demo utilizaré maquinas virtuales para hacer test de los siguientes container runtimes

- docker
- containerd
- crio
- podman
- kata

## Requisitos

- Ubuntu 20.10
- `sudo apt install vagrant`
